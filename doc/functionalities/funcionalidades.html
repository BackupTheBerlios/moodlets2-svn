Este módulo se basará en el Calendario del moodle, el cual se encargara de parsear los eventos de los calendarios y guardarlos en un archivo del tipo iCal.
El archivo principal en el que debemos hacer hincapié es "view.php", el cual se encarga de leer de las bases de datos la información necesaria para mostrar por pantalla los eventos. Esto nos será muy útil ya que reaprovechando la mayoría de este código podemos crear nuestra función que guarde en formato iCal.
Nuestra función principal se encargara de procesar los datos del Calendario y decidirá si escribir el fichero iCal o no, basándose en las preferencias y datos del usuario/curso.
Tan solo necesitaremos una tabla (con sus propias tuplas) para guardar los datos necesarios sobre las preferencias de cada usuario.
